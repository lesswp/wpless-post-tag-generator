== Description ==

WPLess Post Tag Generator is an advanced yet easy-to-use WordPress plugin that automates the process of generating tags for your posts. Instead of manually brainstorming keywords or phrases, this plugin leverages the Gemini API to analyze your post content and generate accurate, contextually relevant, and SEO-friendly tags within seconds.

With WPLess Post Tag Generator, you can specify the exact number of tags to be generated (minimum 5 and up to 10). Once generated, these tags are automatically added to your post, saving you time and ensuring that your content remains optimized for search engines. The plugin seamlessly integrates into the WordPress post editor interface, providing a dedicated "Generate Tags" button and a quantity input field, making tag generation a simple, one-click process.

Additionally, the plugin includes a settings page where you can configure your Gemini API key, ensuring secure and efficient communication with the API. This feature makes the setup process quick and straightforward, even for non-technical users.

Whether you're a blogger, content creator, or SEO enthusiast, WPLess Post Tag Generator helps you maintain a consistent tagging strategy, improving your site's search engine ranking and audience reach. Simplify your content creation workflow and focus more on producing high-quality posts while the plugin handles the tags for you.

== Features ==

- Automatically generates tags based on the content of your posts using the Gemini API.
- Allows you to specify the exact number of tags to generate (minimum 5, up to 10 tags).
- Instantly adds the generated tags to your post, saving you time and effort.
- A "Generate Tags" button is integrated into the WordPress post editor for quick access.
- Provides a settings page for configuring your Gemini API key.
- Easy-to-use interface designed for bloggers, content creators, and SEO professionals.

== Frequently Asked Questions ==

= How do I get a Gemini API key? =

To use the WPLess Post Tag Generator, you'll need a valid Gemini API key. Visit the official Gemini API website (insert website link here), create an account, and follow their instructions to generate an API key. Once you have the key, add it to the plugin's settings page.

= What is the maximum number of tags I can generate? =

The plugin allows you to generate up to 10 tags at a time. You can adjust the quantity using the input field provided next to the "Generate Tags" button in the WordPress post editor. If fewer than 5 tags are generated, the plugin will show an error message to help you try again with better content.

= Can I customize the tags generated? =

The tags are generated automatically based on the content of your post and the advanced language capabilities of the Gemini API. While you can't directly customize the tag generation process, you can edit the tags manually after they are generated.
